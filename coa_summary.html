Project Summary

coa — Coalesce ASCII Hero

An interactive ASCII / “matrix” hero animation built for Coalesce (coalesce.io). White background, blue characters, mouse-reactive. Live

Live: coa.bbase.ai  ·  Folder: ~/Claude/Projects/coa  ·  Repo: bbagent-01/coa

The five modes

Switch with the nav in the top-left. The “content” toggle overlays the Coalesce hero copy on top of any mode.

ModeWhat it does
01 · cursorAn amorphous, constantly-shifting mouse blob with bright-blue (#2962ff) sparkle glints, over a faint light-blue swirl background.
02 · faceA smooth, ethereal 3D face. Blue eyes, a subtle warm/knowing expression, slow blink. Head and eyes follow the cursor slowly (never cross, never jump). Eyes sit level with the “Get Demo” button; edges dissolve into the background.
03 · spiral inData spiralling inward along bent, organic lines toward the center, with a round, solid, bright-blue mouse blob and a long trailing tail riding on top.
04 · face matrixThe same face, but built entirely out of moving code characters with deep blue shading and blue eyes. A “HEAD SIZE” slider scales the head (eyes always stay flanking the headline). Defaults to a ~2× zoom with the eyes on either side of the H1.
05 · spiral followSame as “spiral in,” but the whole spiral slowly drifts so its center moves toward wherever the mouse is.

Hero overlay (“content” toggle)

  • Headline: “Humans and agents delivering data, together.”
  • Subtext, a blue Get Demo button, G2 (4.8) + Gartner (4.9) rating cards, and a client-logo strip.
  • Sits centered over whichever animation is active; the animation still reacts to the mouse behind it.

Under the hood

  • One file: index.html — a static page (vanilla JavaScript + three.js for the 3D face).
  • Deploy: every push to the repo auto-publishes to coa.bbase.ai within ~30 seconds (Cloudflare Pages). No build step.
  • The face is a real 3D head scan rendered live; modes 02 and 04 share the exact same animation, just drawn two different ways (smooth surface vs. code characters).
  • The cursor blob is shared between the cursor and spiral modes — round & solid in the spirals, amorphous in cursor mode.
Performance: “04 · face matrix” is the heaviest mode (it draws a lot every frame). Its background is cached to keep it smooth. If it ever feels slow on a given machine, the fix is to make the characters slightly larger/fewer.

Where it stands & what’s next

Everything above is built and live. Open items / likely next tweaks:
